## How to run

```bash
# 1. Install dependencies
npm install

# 2. Run project locally
npm run dev
```

## Scripts

Use the following scripts for your development workflow:

```bash
# Start the development server
npm run dev

# Build for production
npm run build

# Preview the build
npm run preview
```
